**Recruitment Analytics & Hiring Insights Dashboard (Power BI)**

**Project Overview**:

This project focuses on analyzing recruitment data to evaluate hiring efficiency, identify bottlenecks in the hiring pipeline, and support data-driven decision-making. The dashboard provides a comprehensive view of candidate flow, recruiter performance, and sourcing effectiveness.

**Objectives**:
- Analyze end-to-end recruitment pipeline  
- Measure hiring performance and efficiency  
- Identify bottlenecks in hiring stages  
- Evaluate effectiveness of different hiring sources  
- Assess recruiter performance  

**Dataset**:

The dataset includes candidate-level recruitment data with the following fields:
- Candidate ID  
- Job Role  
- Department  
- Application, Interview, Offer, and Joining Dates  
- Status (Hired, Rejected, Offered)  
- Source (LinkedIn, Referral, Website)  
- Recruiter  
- Salary Offered  

**Tools & Technologies**:

- Power BI: For creating interactive visualizations and dashboards. Excel: For preprocessing and cleaning the dataset.
- DAX (Data Analysis Expressions)  
- Excel (Data Source)  


**Key Features**:

- KPI cards for total candidates, hires, hiring rate, and average time-to-hire  
- Hiring funnel to visualize candidate drop-off across stages  
- Source effectiveness analysis  
- Department-wise hiring insights  
- Recruiter performance analysis  
- Offer vs acceptance comparison  

**Key Insights**: 

- Referral-based candidates show higher hiring success compared to other sources  
- Significant candidate drop-off occurs at the interview stage  
- IT department has the highest hiring volume  
- Recruiter performance varies, with some recruiters consistently achieving higher conversions  
- Not all offers result in hires, indicating potential gaps in offer acceptance
